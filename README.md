## ðŸŽ¨ Customization Guide

### Update Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary: #667eea;
    --accent: #f6ad55;
    --dark: #0f172a;
}
```

### Modify Content
All content in `index.html` - no template engine needed

### Adjust Animations
Configure AOS in `script.js`:
```javascript
AOS.init({
    duration: 800,
    easing: 'ease-in-out',
    once: true
});
```
